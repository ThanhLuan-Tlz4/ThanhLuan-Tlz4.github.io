During my internship at **Amazon Web Services Vietnam Co., Ltd.** from **17/04/2026** to **30/07/2026**, I had the opportunity to learn, practice, and apply the knowledge I had gained at school to a real working environment.

I participated in researching, implementing, and completing the **Secure AI-Driven Document Platform system based on a Serverless architecture on AWS**, taking responsibility for building the Backend, **designing APIs, deploying Cloud infrastructure, and integrating AWS services such as Amazon S3, Amazon DynamoDB, AWS Lambda, Amazon API Gateway, Amazon Cognito, Amazon GuardDuty, and Amazon Bedrock to build an automated document upload, security scanning, and document analysis workflow.**

Through the internship process, I improved the following skills:

* Designing Serverless system architecture on AWS.
* Developing Backend using AWS Lambda and Amazon API Gateway.
* Designing and building RESTful APIs.
* Managing data with Amazon DynamoDB.
* Building a file Upload mechanism using Amazon S3 Presigned URLs.
* Managing user authentication and authorization with Amazon Cognito and IAM.
* Integrating AWS services such as Amazon GuardDuty, Amazon Bedrock, and Amazon CloudWatch.
* Testing, logging, and handling errors in a Cloud environment.
* Designing infrastructure and managing resources on AWS.
* Analyzing requirements, designing systems, and solving problems.
* Working in a team, managing work progress, and coordinating project development.
* Writing technical documentation and reports on the system implementation process.

Regarding working attitude, I always tried to complete assigned tasks well, proactively researched AWS documentation, followed the team’s working process, actively communicated with team members to resolve issues arising during system development, and continuously learned to improve my professional knowledge and skills.

## General Evaluation

**1. Learning and Working Environment**

The AWS First Cloud Journey program provides a professional, dynamic learning environment that encourages self-learning. The training content is built with a clear roadmap, from foundational knowledge to advanced topics, helping learners gradually approach Amazon Web Services in a structured way. In addition, the hands-on Workshops give learners the opportunity to deploy directly in the AWS environment instead of only studying theory.

**2. Support from Mentors and the Organizing Team**

Mentors are always enthusiastic in answering questions during the learning process and Capstone Project implementation. Instead of giving direct answers immediately, mentors often guide learners on how to approach problems so they can research and solve issues by themselves, helping develop research ability and technical thinking.

The organizing team also regularly updates documents, announcements, and supports learners when they encounter issues related to AWS accounts or Workshop content.

**3. Relevance Between Program Content and Major**

The program content is suitable for the Information Technology major, especially for directions related to Cloud Computing, Backend Development, and DevOps. The knowledge learned is not only theoretical but also closely connected to hands-on exercises and real projects, helping learners clearly understand how to apply AWS services in system development.

**4. Learning Opportunities and Skill Development**

Through the program, I had the opportunity to access many AWS services such as Amazon EC2, Amazon VPC, Amazon S3, Amazon DynamoDB, AWS Lambda, Amazon API Gateway, Amazon Cognito, Amazon GuardDuty, and Amazon Bedrock.

In addition to professional knowledge, I also developed many skills such as:

* Requirements analysis and system design.
* Designing Serverless architecture on AWS.
* Teamwork and project progress management.
* Reading and understanding English technical documentation.
* Self-researching new technologies and solving problems.
* Presenting, reporting, and writing technical documentation.

**5. Learning Culture and Supportive Spirit**

What I appreciate the most is the supportive spirit between mentors and learners in the program. When facing difficulties during practice or project implementation, everyone is always willing to exchange experiences and find solutions together. This creates a positive learning environment, giving learners more motivation to complete the Workshops and Capstone Project.

**6. Training Program**

The learning roadmap is designed reasonably, starting from basic AWS knowledge, then expanding to Networking, Storage, Security, Serverless, and finally the Capstone Project.

The combination of theory, Workshops, and real projects helps learners not only understand how to use AWS services but also know how to combine multiple services to build a complete system.

## Other Questions

### What were you most satisfied with during your participation in the program?

What I was most satisfied with was being able to participate in a highly practical program. Instead of learning each service separately, the program created an opportunity for learners to build a complete project on AWS. This helped me better understand how AWS services work together in a real system and significantly improved my Cloud deployment skills.

### What do you think the program needs to improve?

In my opinion, the program could add more in-depth sharing sessions on real-world AWS architectures such as Microservices, CI/CD, Infrastructure as Code, or FinOps. In addition, more Code Review sessions or Debug Workshops could be organized so learners can gain more experience in handling common situations during system development.

### If you introduced the program to your friends, would you recommend them to join? Why?

I would be willing to introduce the AWS First Cloud Journey program to my friends because it is a high-quality program that combines theory and practice effectively. After completing the program, learners not only understand AWS services but also have the opportunity to build a real project, thereby creating a strong foundation for studying and working in the field of Cloud Computing.

## Suggestions and Expectations

I hope the program will continue to update Workshops according to new AWS services and features, while also adding more practical exercises so learners can approach enterprise-level problems.

In addition, if there are more experience-sharing sessions from AWS Solutions Architects, DevOps Engineers, or alumni who are working in the Cloud field, it will help learners gain more practical perspectives and clearer career orientation.

If there is an opportunity in the future, I would like to continue participating in advanced AWS training programs, especially topics related to Serverless Architecture, AI on AWS, DevOps, and Security.

Finally, I would like to express my sincere thanks to the mentors and the AWS First Cloud Journey organizing team for building a high-quality learning program, creating opportunities for me to access modern Cloud technologies, improve my professional knowledge, and develop many essential skills for my future work.

This section introduces the technical blogs that I have published on the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Each article summarizes a specific AWS topic, ranging from Identity and Access Management (IAM) to Serverless architecture and cloud security monitoring.

### [Blog 1 - IDENTITY AND ACCESS MANAGEMENT ON AWS](3.1-Blog1/)
This blog provides an overview of AWS Identity and Access Management (IAM), covering IAM Identity Center, Service Control Policies (SCPs), Resource Control Policies (RCPs), Permissions Boundaries, and the relationship between identity-based and resource-based policies for implementing a defense-in-depth security model.

### [Blog 2 - ASSUME ROLE AND SESSION TAGS FOR SECURE AWS ACCESS MANAGEMENT](3.2-Blog2/)
This blog explains advanced IAM access control mechanisms, including Assume Role, IAM Conditions, AWS Security Token Service (STS), and Session Tags. It also introduces how AWS IAM Identity Center enables Attribute-Based Access Control (ABAC) to simplify permission management in multi-account environments.

### [Blog 3 - BUILDING SERVERLESS APPLICATIONS AND MONITORING SECURITY ON AWS](3.3-Blog3/)
This blog introduces the fundamentals of building serverless applications using AWS Lambda, Amazon S3, and Amazon DynamoDB. It also explores security monitoring solutions such as GoAccess, Fail2Ban, and Wazuh to improve visibility, automate threat detection, and strengthen cloud security.

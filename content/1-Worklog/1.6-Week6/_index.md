### Week 6 Objectives:

* Understand storage and data protection services on AWS.
* Learn about Amazon S3, AWS Storage Gateway, Snow Family, Disaster Recovery, and AWS Backup.
* Understand the process of backing up, restoring data, and migrating virtual machines to AWS.
* Practice deploying AWS Backup and VM Import/Export.
* Enhance knowledge through the AWS Event Meeting and experience sharing with the community.

### Week 6 Achievements:

* Understood the role of Amazon S3 in storing Object-based data and the relationship between Amazon S3, AWS Storage Gateway, AWS Backup, and Disaster Recovery solutions within the AWS ecosystem.

* Understood the architecture and use cases of Amazon Storage Gateway, Snow Family, and Disaster Recovery on AWS, thereby understanding how AWS supports Hybrid Cloud models and enterprise data protection.

* Successfully deployed AWS Backup and created a Backup Vault, Backup Plan, and Resource Assignment to automatically back up AWS resources according to the configured policy.

* Successfully performed resource Backup and Restore through AWS Backup, checked the Recovery Point, and verified data recovery capability after an incident.

* Successfully practiced VM Import/Export by preparing a virtual machine from an On-premises environment, importing it as an Amazon Machine Image (AMI), and launching an Amazon EC2 Instance from the converted AMI.

* Verified the integrity of the operating system and configuration after the Import process, thereby understanding how to migrate traditional infrastructure to AWS without reinstalling the system.

* Evaluated the advantages of VM Import/Export in the Migration process, allowing existing virtual machines to be reused for deployment on AWS while reducing migration time and costs.

* Completed all hands-on exercises and cleaned up resources after finishing the Labs to ensure that the AWS environment did not incur unnecessary costs.

* Participated in the AWS Event Meeting and exchanged knowledge with the community about AWS learning methods, Hackathon participation experience, teamwork skills, overcoming lack of confidence and procrastination in learning, while also gaining practical insights into product development on the AWS platform.

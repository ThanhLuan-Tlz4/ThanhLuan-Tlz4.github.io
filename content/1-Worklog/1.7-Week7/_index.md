### Week 7 Achievements:

* Reviewed and organized the knowledge learned about Amazon S3, AWS Storage Gateway, AWS Backup, VM Import/Export, and Disaster Recovery, thereby gaining a clearer understanding of the relationship between storage and data protection services on AWS.

* Successfully deployed Amazon FSx for Windows File Server, created a File System, and configured the necessary network components so that the service could operate within Amazon VPC.

* Connected Amazon EC2 to Amazon FSx through the SMB protocol, created a Shared Folder, and tested the ability to read, write, and share data between machines within the same system.

* Understood how Amazon FSx provides a fully managed File Server service on AWS, reducing operational effort compared to deploying a traditional Windows File Server independently.

* Practiced creating and managing an Amazon S3 Bucket and performed Upload, Download, deletion, and Object management through the AWS Management Console.

* Successfully configured Static Website Hosting, Bucket Policy, and Block Public Access, while also testing access to the Website Endpoint after the configuration was completed.

* Practiced Versioning on Amazon S3 to track Object versions, thereby understanding the data protection mechanism against accidental overwrites or deletions.

* Compared the differences between Amazon S3 (Object Storage), Amazon FSx (Managed File Storage), AWS Storage Gateway (Hybrid Storage), and AWS Backup (Data Protection), while identifying suitable real-world use cases for each service.

* Completed all hands-on exercises and cleaned up resources after finishing the Workshop to ensure that no additional costs were incurred on the AWS account.

### Week 4 Achievements:

* Fully completed Workshop 19 and Workshop 20 in the AWS First Cloud Journey program.

* Practiced deploying and configuring AWS resources according to the instructions of each Workshop, while also verifying the operation of the system after completion.

* Understood the process of deploying, configuring, and validating AWS services in a practical environment.

* Strengthened skills in using the AWS Management Console and became familiar with the process of completing Lab exercises.

* Reviewed and understood the roles of important Compute and Storage services on AWS, including:

  * Amazon Elastic Compute Cloud (EC2).
  * Amazon Lightsail.
  * Amazon Elastic File System (EFS).
  * Amazon FSx.
  * AWS Application Migration Service (MGN).

* Distinguished the purpose of each service:

  * EC2 is used to deploy highly customizable virtual servers.
  * Lightsail is suitable for small applications and rapid deployment.
  * EFS provides a shared file system for Linux.
  * FSx provides specialized file systems for various requirements.
  * AWS MGN supports migrating servers from On-premises environments to AWS.

* Developed the ability to select appropriate Compute and Storage services for different deployment requirements and became prepared for Hybrid Cloud and Migration topics in the following weeks.

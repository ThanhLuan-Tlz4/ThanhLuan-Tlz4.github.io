Trong suốt **12 tuần** tham gia chương trình **AWS First Cloud Journey**, tôi đã từng bước tiếp cận và thực hành các dịch vụ cốt lõi của Amazon Web Services thông qua hệ thống bài giảng lý thuyết, Workshop thực hành và đồ án cuối kỳ. Quá trình học tập không chỉ giúp tôi nắm vững kiến thức nền tảng về điện toán đám mây mà còn có cơ hội áp dụng trực tiếp các dịch vụ AWS vào việc xây dựng một hệ thống thực tế theo kiến trúc Serverless.

Thông qua các Workshop và Capstone Project, tôi đã thực hành triển khai nhiều dịch vụ AWS như Amazon EC2, Amazon VPC, Amazon S3, Amazon DynamoDB, AWS Lambda, Amazon API Gateway, Amazon Cognito, AWS Backup, AWS Storage Gateway, Amazon FSx, AWS Security Hub, AWS KMS, IAM, CloudWatch cùng nhiều dịch vụ khác. Đồng thời, tôi cũng được tiếp cận các nguyên tắc thiết kế kiến trúc Cloud theo AWS Well-Architected Framework, tối ưu chi phí, bảo mật và khả năng mở rộng của hệ thống.

Trong giai đoạn cuối chương trình, nhóm đã xây dựng thành công một dự án Serverless trên AWS, qua đó giúp tôi hiểu rõ quy trình từ phân tích yêu cầu, thiết kế kiến trúc, triển khai hạ tầng, phát triển Backend, kiểm thử, tối ưu hệ thống đến hoàn thiện sản phẩm.

### Tổng kết các tuần thực hiện

**Tuần 1:** [Làm quen với AWS và các dịch vụ cơ bản](1.1-week1/)

- Làm quen với chương trình AWS First Cloud Journey.
- Tạo tài khoản AWS và tìm hiểu AWS Management Console.
- Cài đặt AWS CLI và cấu hình môi trường làm việc.
- Tìm hiểu các nhóm dịch vụ cơ bản của AWS.

**Tuần 2:** [IAM và quản lý quyền truy cập](1.2-week2/)

- Tìm hiểu AWS IAM.
- Quản lý User, Group, Role và Policy.
- Thực hành phân quyền theo nguyên tắc Least Privilege.

**Tuần 3:** [Amazon VPC và Networking](1.3-week3/)

- Xây dựng hệ thống mạng trên AWS.
- Thực hành VPC, Subnet, Route Table, Internet Gateway.
- Tìm hiểu Security Group và Network ACL.

**Tuần 4:** [Infrastructure as Code với AWS CloudFormation](1.4-week4/)

- Tìm hiểu Infrastructure as Code.
- Thực hành xây dựng và triển khai CloudFormation Template.
- Quản lý hạ tầng bằng Stack.

**Tuần 5:** [AWS Storage và Backup](1.5-week5/)

- Thực hành Amazon S3.
- Triển khai AWS Backup.
- Cấu hình AWS Storage Gateway.
- Hiểu mô hình Hybrid Storage.

**Tuần 6:** [Storage Services và Migration](1.6-week6/)

- Học lý thuyết về Amazon S3, Storage Gateway, Snow Family và Disaster Recovery.
- Triển khai AWS Backup.
- Thực hành VM Import/Export.
- Tham gia AWS Event Meeting và trao đổi kinh nghiệm học tập.

**Tuần 7:** [Amazon FSx và Amazon S3](1.7-week7/)

- Ôn tập các dịch vụ lưu trữ AWS.
- Triển khai Amazon FSx for Windows File Server.
- Thực hành quản lý dữ liệu trên Amazon S3.
- So sánh các giải pháp lưu trữ trên AWS.

**Tuần 8:** [AWS Security và Cost Optimization](1.8-week8/)

- Học về IAM, Cognito, Organizations và AWS KMS.
- Thực hành AWS Security Hub.
- Tối ưu chi phí Amazon EC2 bằng AWS Lambda.
- Quản lý tài nguyên bằng Tags và Resource Groups.

**Tuần 9:** [IAM nâng cao và AWS KMS](1.9-week9/)

- Quản lý quyền truy cập EC2 bằng Resource Tags.
- Thực hành IAM Permission Boundary.
- Mã hóa dữ liệu bằng AWS KMS.
- Áp dụng nguyên tắc Least Privilege trong phân quyền.

**Tuần 10:** [Chuẩn bị Capstone Project](1.10-week10/)

- Xác định chủ đề đồ án cuối kỳ.
- Phân tích yêu cầu hệ thống.
- Thiết kế kiến trúc AWS tổng quát.
- Phân chia nhiệm vụ cho các thành viên trong nhóm.

**Tuần 11:** [Triển khai Backend Serverless](1.11-week11/)

- Xây dựng hạ tầng Backend trên AWS.
- Triển khai Amazon S3, DynamoDB, Lambda và API Gateway.
- Xây dựng REST API và cơ chế Upload bằng Presigned URL.
- Tích hợp Cognito, GuardDuty, Bedrock và CloudWatch.

**Tuần 12:** [Kiểm thử và hoàn thiện hệ thống](1.12-week12/)

- Kiểm thử toàn bộ hệ thống.
- Khắc phục lỗi và tối ưu Backend.
- Kiểm thử tích hợp các dịch vụ AWS.
- Hoàn thiện tài liệu và chuẩn bị báo cáo đồ án.

## Những kiến thức và kỹ năng đạt được

Sau khi hoàn thành chương trình AWS First Cloud Journey, tôi đã đạt được những kết quả sau:

- Hiểu được các khái niệm nền tảng về Cloud Computing và hệ sinh thái dịch vụ của Amazon Web Services.
- Có khả năng thiết kế kiến trúc hệ thống trên AWS theo mô hình Cloud và Serverless.
- Thành thạo triển khai các dịch vụ như Amazon S3, Amazon EC2, Amazon DynamoDB, AWS Lambda, API Gateway, Amazon Cognito, CloudWatch và IAM.
- Hiểu và áp dụng các nguyên tắc bảo mật trên AWS như IAM Policy, Resource Tags, Permission Boundary, AWS Security Hub và AWS KMS.
- Biết xây dựng RESTful API bằng AWS Lambda kết hợp API Gateway.
- Triển khai cơ chế Upload tệp trực tiếp lên Amazon S3 thông qua Presigned URL.
- Thực hành xây dựng hệ thống lưu trữ và quản lý dữ liệu bằng Amazon DynamoDB.
- Hiểu quy trình Backup, Restore, Migration và Hybrid Storage trên AWS.
- Có khả năng tích hợp nhiều dịch vụ AWS để xây dựng một hệ thống hoàn chỉnh.
- Biết sử dụng Amazon CloudWatch để theo dõi, ghi log và hỗ trợ Debug hệ thống.
- Rèn luyện kỹ năng làm việc nhóm, phân tích yêu cầu, thiết kế hệ thống, lập kế hoạch và triển khai dự án thực tế.
- Hoàn thành thành công Capstone Project theo kiến trúc Serverless, áp dụng các kiến thức đã học trong suốt chương trình để xây dựng một hệ thống có khả năng mở rộng, bảo mật và dễ bảo trì.

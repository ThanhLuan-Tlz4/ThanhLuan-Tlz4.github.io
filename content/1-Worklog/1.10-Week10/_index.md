### Week 10 Achievements:

* Completed the team meeting and agreed on the final project topic, clearly defining the objectives, implementation scope, and main functions of the system.

* Analyzed the project requirements and selected AWS services suitable for the system architecture, ensuring that the requirements for data storage, business logic processing, user management, security, and monitoring were met.

* Evaluated the role of each AWS service in the system, such as Amazon S3 for data storage, AWS Lambda for Serverless business logic processing, Amazon API Gateway for providing REST APIs, Amazon DynamoDB for storing NoSQL data, and Amazon CloudWatch for system monitoring.

* Completed the overall system architecture diagram on AWS, describing the data processing flow between components and how AWS services work together to operate the entire application.

* Discussed and evaluated the architecture based on scalability, availability, security, and cost optimization criteria, thereby selecting an implementation approach suitable for the project requirements.

* Assigned specific tasks to each team member across development areas such as Frontend, Backend, and AWS infrastructure deployment, helping establish a clear project execution plan and improve team coordination.

* Developed an implementation plan and roadmap for the project development phase, creating a foundation for building the system in the following weeks.

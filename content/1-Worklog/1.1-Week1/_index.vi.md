### Kết quả đạt được tuần 1:

* Hiểu được khái niệm điện toán đám mây (Cloud Computing) và tổng quan về Amazon Web Services (AWS).

* Nắm được các nhóm dịch vụ chính của AWS, bao gồm:
  * Compute
  * Storage
  * Networking
  * Database
  * Security & Identity
  * Monitoring & Management

* Hoàn thành việc tạo và cấu hình tài khoản AWS phục vụ cho quá trình học tập và thực hành.

* Làm quen với giao diện AWS Management Console và biết cách tìm kiếm, truy cập cũng như quản lý các dịch vụ trên nền tảng AWS.

* Cài đặt và cấu hình thành công AWS CLI trên máy tính, bao gồm:
  * Access Key
  * Secret Access Key
  * Default Region
  * Output Format

* Sử dụng AWS CLI để thực hiện các thao tác cơ bản như:
  * Kiểm tra thông tin tài khoản AWS.
  * Kiểm tra cấu hình CLI.
  * Liệt kê các AWS Region.
  * Xem thông tin EC2.
  * Tạo và quản lý EC2 Key Pair.
  * Kiểm tra các tài nguyên AWS đang hoạt động.

* Hoàn thành các bài thực hành đầu tiên trên nền tảng First Cloud AI Journey sử dụng AWS Promotional Credit (100 USD), đồng thời nắm được cách quản lý tài nguyên nhằm tối ưu chi phí và tránh phát sinh ngoài dự kiến.

* Có khả năng sử dụng kết hợp AWS Management Console và AWS CLI để quản lý các tài nguyên AWS, tạo nền tảng cho các nội dung nâng cao trong những tuần tiếp theo.

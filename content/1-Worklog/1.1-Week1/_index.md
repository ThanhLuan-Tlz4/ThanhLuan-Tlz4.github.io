### Week 1 Objectives:

* Connect and become familiar with the members of the First Cloud AI Journey program.
* Learn about the rules, learning process, and working procedures of the program.
* Create an AWS account and prepare the practice environment.
* Understand the basic service categories of Amazon Web Services (AWS), how to use the AWS Management Console, and AWS CLI.
* Become familiar with hands-on exercises using AWS Promotional Credit (100 USD).

### Week 1 Achievements:

* Understood the concept of Cloud Computing and gained an overview of Amazon Web Services (AWS).

* Understood the main AWS service categories, including:

  * Compute
  * Storage
  * Networking
  * Database
  * Security & Identity
  * Monitoring & Management

* Completed the creation and configuration of an AWS account for learning and practice purposes.

* Became familiar with the AWS Management Console interface and learned how to search for, access, and manage services on the AWS platform.

* Successfully installed and configured AWS CLI on the computer, including:

  * Access Key
  * Secret Access Key
  * Default Region
  * Output Format

* Used AWS CLI to perform basic operations such as:

  * Checking AWS account information.
  * Checking CLI configuration.
  * Listing AWS Regions.
  * Viewing EC2 information.
  * Creating and managing EC2 Key Pairs.
  * Checking active AWS resources.

* Completed the first hands-on exercises on the First Cloud AI Journey platform using AWS Promotional Credit (100 USD), while also understanding how to manage resources to optimize costs and avoid unexpected charges.

* Developed the ability to use both the AWS Management Console and AWS CLI to manage AWS resources, establishing a foundation for more advanced content in the following weeks.

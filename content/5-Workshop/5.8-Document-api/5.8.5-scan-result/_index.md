In this step, we will use the `GetDocumentScanResultLambda` function to return the complete scanning and evaluation status of a document.

The Lambda function is integrated with the following API route:

```text
GET /documents/{documentId}/scan-result
```

Configure the following environment variable:

```text
DOCUMENTS_TABLE = SecureDocuments
```

The Lambda function receives the `documentId`, retrieves the corresponding item from the `SecureDocuments` table, and verifies that the document's `userId` matches the `sub` claim in the JWT token.

The Lambda execution role requires the following permission:

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "ReadDocumentScanResult",
      "Effect": "Allow",
      "Action": [
        "dynamodb:GetItem"
      ],
      "Resource": "arn:aws:dynamodb:ap-southeast-1:950725740411:table/SecureDocuments"
    }
  ]
}
```

The response is organized into the following sections:

```text
processing
malwareScan
aiAnalysis
systemDecision
review
finalResult
```

The `processing` section indicates the current processing stage. The Lambda function maps document statuses as follows:

```text
PENDING_UPLOAD         → UPLOAD
WAITING_MALWARE_SCAN   → MALWARE_SCAN
AI_ANALYZING           → AI_ANALYSIS
AI_ANALYSIS_COMPLETED  → DECISION
MOVING_TO_CLEAN        → FINALIZING
MOVING_TO_REJECT       → FINALIZING
MANUAL_REVIEW          → MANUAL_REVIEW
SAFE                   → COMPLETED
REJECTED               → COMPLETED
AI_ERROR               → ERROR
SCAN_ERROR             → ERROR
```

The `malwareScan` section returns:

```text
status
result
threats
statusReasons
scannedAt
```

The `aiAnalysis` section returns:

```text
status
provider
model
riskScore
riskLevel
recommendedAction
confidence
summary
categories
phishingDetected
socialEngineeringDetected
credentialTheftDetected
promptInjectionDetected
suspiciousLinks
completedAt
```

The `systemDecision` section returns the calculated risk score, risk level, recommended action, and the reason generated by the system.

The `review` section contains the administrator's manual review results.

The `finalResult` section returns the final decision and determines whether the document can be downloaded:

```python
downloadAvailable = (
    item.get("status") == "SAFE"
    and item.get("finalDecision") == "ALLOW"
    and item.get("currentPrefix") == "clean"
)
```

Send the following request:

```http
GET https://u0figy4o19.execute-api.ap-southeast-1.amazonaws.com/documents/<DOCUMENT_ID>/scan-result
Authorization: Bearer <ACCESS_TOKEN>
```
